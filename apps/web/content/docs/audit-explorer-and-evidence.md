## Audit Search Basics
Use `/dashboard/admin/audit` filters by entity, action, store ID, and actor user ID.

Start broad, then narrow filters to isolate the event sequence for the incident timeline.

## Evidence Export Playbook
Build evidence packs with:

- Audit event IDs and timestamps
- Actor and store attribution
- Related notification/acceptance records

Reference runbooks in `docs/runbooks/legal-update-communications.md` and `docs/runbooks/admin-incident-escalation.md` for response standards.

## Validation Steps
Cross-check audit entries against operational panels (stores, moderation, legal) before issuing a final response.
